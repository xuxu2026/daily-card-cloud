# MEMORY.md - 项目长期记忆

## 技术要点

- **图片渲染**: `card_generator.py` 使用 Playwright Python API + `channel='msedge'` 启动系统 Edge 浏览器进行无头截图，无需下载 Chromium
- **Windows 控制台编码**: main.py 开头加了 `sys.stdout = io.TextIOWrapper(..., encoding='utf-8')` 永久解决 GBK 问题

## 风格系统（72+24 独立节日主题）

### 常规主题（72种）
- **编号**: S01-S72
- **规则**: 按一年中第几天轮换，两个多月不重样

  - D 系 薰衣草紫（28-36）: 优雅神秘，梦幻浪漫
  - E 系 玫瑰粉（37-45）: 柔美优雅，女性气质

  - H 系 琥珀金（58-72）: 温暖贵气，秋日暖阳

### 节日主题（32种，扩展版）
- **编号**: H01-H32
- **规则**: 节日期间优先使用，独立于72种常规主题之外
- **设计原则**: 全部精致浅色系，避免暗色背景、高饱和红、红字
- **配色关联**: 每种节日主题颜色与节日本身强关联（如春节红金、元宵暖黄、端午艾绿、中秋金黄、圣诞红绿等）
- **暗纹关联**: 每种节日主题配专属节日暗纹（如春节金币/灯笼、端午粽叶、中秋月亮、万圣南瓜、圣诞雪花等）
- **包含**: 春节(2套)、元宵(2套)、清明(2套)、端午(2套)、七夕(2套)、中秋(2套)、国庆(2套)、重阳(2套)、万圣(2套)、感恩(2套)、圣诞(2套)、跨年/元旦(2套)、情人节(2套)、妇女节、劳动节、儿童节(2套)、教师节(2套)

### 暗纹系统
- **card_generator.py** 中内置 48 种 SVG 暗纹装饰（30种通用 + 18种节日专属）
- **通用暗纹** (30种): roses, cherry, lavender, olive, fog, bamboo, moss, waves, grass, fern, mountains, lines, inkwash, neon, filmstrip, dots_clean, clouds_soft, petals, leaves_minimal, waves_minimal, triangle, circuit, window, barcode, brushstroke, wave_ukiyoe, grid, aurora, lavender_field, maple, cream
- **节日专属暗纹** (18种): lantern(春节), sparkle(星光), star(国庆), snow(圣诞/元旦), pumpkin(万圣), balloon(儿童节), sunflower(教师节), grain(麦穗), chrysanthemum(重阳), moon(中秋), heart(七夕), rain(清明), bamboo_leaf(竹叶), cloud(云), zongye(端午), coin(春节), butterfly(蝴蝶), halo(光环)
- **72种通用主题**: 各配不同暗纹，共覆盖 28 种装饰类型，避免审美疲劳
- **32种节日主题**: 各配专属节日暗纹，颜色与节日强关联

### 背景方案
- **全部预设**: 背景/配色/装饰全部预设，不使用网搜图片
- **WCAG AA 对比度**: 所有主题文字与背景对比度符合 AA 标准
- **渐变色**: 每种风格预设 5 阶渐变背景（160deg）
- **卡片**: 半透明白色背景 + 柔和阴影 + 圆角
